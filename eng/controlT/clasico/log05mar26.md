
# Retroalimentacion

* r → [+] → [G] → y
(con b ↑ desde [H]) e = r + b   Positiva (amplifica)


* r → [-] → [G] → y
(con b ↑ desde [H]) e = r - b   Negativa (estabiliza)

Positiva [+]:  Y(s)/R(s) = G(s) / (1 - G(s)H(s))
Negativa [-]:  Y(s)/R(s) = G(s) / (1 + G(s)H(s))

Positiva: Si |G·H| > 1, denominador → 0 → inestabilidad/oscilaciones

Negativa: Denominador > 1 → ganancia reducida pero estable.


## Efectos Prácticos

[+] Positiva: Aumenta ganancia, pero sensible a perturbaciones y propensa a inestabilidad. Útil en osciladores.



[-] Negativa: Reduce error de estado estacionario, rechaza perturbaciones, mejora estabilidad. Estándar en control automático.


El [-] es el convencional para la mayoría de aplicaciones de ingeniería de control.
